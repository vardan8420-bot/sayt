Шаг 1: Создайте репозиторий на GitHub
  → Откройте: https://github.com/new
  → Repository name: sayt (или любое имя)
  → Description: "Modern Marketplace Platform"
  → Выберите Public или Private
  → НЕ ставьте галочку "Add a README file"
  → Нажмите "Create repository"

Шаг 2: Подключите remote (в PowerShell):
  git remote add origin https://github.com/YOUR_USERNAME/sayt.git
  (замените YOUR_USERNAME на ваш GitHub username)

Шаг 3: Загрузите код:
  git push -u origin main
